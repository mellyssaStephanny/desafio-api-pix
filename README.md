# Como executar o projeto

```bash
# Clonando o repositório
git clone https://github.com/mellyssaStephanny/desafio-api-pix.git

# Entre na pasta do projeto
$ cd myapp

# Instale as dependências
$ bundle install

# Execute o servidor
$ rails s
```

# Rotas

- POST/chaves
  - criar chave
- GET/chaves
  - listar todas chaves
- DEL/chaves/:id
  - apagar chave
- PUT/chaves/:id
  - atualizar chave
